## 通用说明

### 请求头
所有接口建议携带：
- `Content-Type: application/json`
- `Authorization: Bearer {token}`（如网关开启鉴权）

### 响应格式
当前服务统一返回以下结构（`DefaultDataResp` / `DefaultMsgResp`）：

```typescript
interface DefaultDataResp<T> {
  code: number       // 200成功，非200失败
  success: boolean   // true成功，false失败
  message: string    // 描述信息
  data: T            // 业务数据
}

interface DefaultMsgResp {
  code: number
  success: boolean
  message: string
  msg: string
}
```

### 状态码说明
- `200`：请求成功
- `500`：请求失败（包括参数错误、业务校验失败、内部异常）

**接口地址：** `GET /api/v1/road-statuses`

**请求参数：**

