# 4.1.2 断面参数检测接口

接口路径：`GET /api/v1/section/detection/parameters`

## 请求参数
| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `timestamp` | long | 是 | 毫秒时间戳 |

## 返回示例
```json
{
  "code": 200,
  "success": true,
  "message": "断面参数检测数据",
  "data": {
    "timestamp": 1711958400000,
    "recordList": [
      {
        "direction": "哈密",
        "startLocation": 0,
        "endLocation": 1000,
        "startStake": "K3178",
        "endStake": "K3179",
        "currentVehicleCount": 51,
        "congestionStatus": "通畅",
        "averageSpeed": 80.0,
        "segmentType": "路段"
      }
    ]
  }
}
```
