# 运维后台接口文档

- 模块：`screen-web-sx`
- 基础路径：`/api/operation-maintenance`
- 认证方式：JWT Bearer Token

统一响应结构：
```json
{
  "code": 200,
  "message": "ok",
  "data": {}
}
```

## 3.2 右侧功能区域

### 3.2.1 车辆信息审核列表
接口：`GET /api/operation-maintenance/vehicle-audit`

Query 参数：
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|
| `pageNo` | number | 否 | 1 | 页码（从 1 开始） |
| `pageSize` | number | 否 | 10 | 每页条数（最大 200） |
| `keyword` | string | 否 | - | 车辆名称/车牌模糊匹配 |
| `auditStatus` | string | 否 | - | 审核状态：`unaudited` / `passed` / `rejected` |

返回 `data` 示例：
```json
{
  "pendingCount": 12,
  "page": {
    "pageNo": 1,
    "pageSize": 10,
    "total": 20,
    "records": [
      {
        "id": 1,
        "vehicleName": "我在录音",
        "licensePlate": "新CZ8634",
        "auditStatus": "passed",
        "auditStatusText": "approved",
        "currentPoints": 12,
        "createTime": "2026-04-01T08:12:00"
      }
    ]
  }
}
```

### 3.2.2 车辆审核明细
接口：`GET /api/operation-maintenance/vehicle-audit/{licensePlate}`

功能说明：
- 展示车辆完整审核信息（号牌、车辆类型、所有人、使用性质、品牌型号、VIN、发动机号、注册日期、行驶证照片等）
- 返回审核勾选项状态（`checkItems`）
- 返回预约统计（`reservationStats`）和最近预约（`latestReservation`）

返回 `data` 示例：

### 3.2.3 审核操作
1) 审核通过
- 接口：`POST /api/operation-maintenance/vehicle-audit/{licensePlate}/approve`
- 请求体：无

2) 审核拒绝
- 接口：`POST /api/operation-maintenance/vehicle-audit/{licensePlate}/reject`
- 请求体：
```json
{
  "rejectItems": ["plate", "vin"],
  "rejectRemark": "photo is blurry"
}
```

`rejectItems` 可选 `key`：
- `plate`
- `vehicleType`
- `owner`
- `usageNature`
- `brandModel`
- `vin`
- `engineNumber`
- `registerDate`
- `other`

### 3.2.4 审核结果导出
接口：`GET /api/operation-maintenance/vehicle-audit/export`

Query 参数：
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|
| `keyword` | string | 否 | - | 车辆名称/车牌模糊匹配 |
| `auditStatus` | string | 否 | - | 审核状态：`unaudited` / `passed` / `rejected` |

返回：CSV 文件下载
- 文件名：`vehicle-audit-export.csv`
- 列：`vehicleName,licensePlate,auditStatus,auditStatusText,currentPoints,createTime,rejectReason`

## 3.3 左侧功能区域

### 3.3.1 在途车辆实时监管
接口：`GET /api/operation-maintenance/realtime-vehicles`

Query 参数：
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|
| `pageNo` | number | 否 | 1 | 页码（从 1 开始） |
| `pageSize` | number | 否 | 10 | 每页条数（最大 200） |
| `vehicleId` | string | 否 | - | 车辆 ID 精准检索 |
| `licensePlate` | string | 否 | - | 车牌精准检索 |
| `direction` | string | 否 | - | 行驶方向 |

返回 `data` 示例：
```json
{
  "pageNo": 1,
  "pageSize": 10,
  "total": 1,
  "records": [
    {
      "id": "12345678901",
      "currentLocation": "K3154+021",
      "licensePlate": "新A66666",
      "vehicleType": "小型客车",
      "speedKmh": 116,
      "direction": "hamimi_to_tuyugou",
      "reportTime": "2026-04-01T10:23:00"
    }
  ]
}
```

### 3.3.2 预约出行信息
接口：`GET /api/operation-maintenance/reservations`

Query 参数：
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|
| `pageNo` | number | 否 | 1 | 页码（从 1 开始） |
| `pageSize` | number | 否 | 10 | 每页条数（最大 200） |
| `licensePlate` | string | 否 | - | 车牌检索 |
| `startTime` | string | 否 | - | 创建时间下限（ISO-8601） |
| `endTime` | string | 否 | - | 创建时间上限（ISO-8601） |
| `deductedOnly` | boolean | 否 | false | 是否仅返回扣分车辆 |

返回 `data` 示例：
```json
{
  "pageNo": 1,
  "pageSize": 10,
  "total": 1,
  "records": [
    {
      "id": 2001,
      "licensePlate": "新A77777",
      "travelTimeSlot": "2026-04-01 18:30-19:30",
      "vehicleType": "小型客车",
      "deductionRecord": "deduct 3 points, remain 9",
      "createTime": "2026-04-01T09:45:00"
    }
  ]
}
```

## 错误码说明
- `400`：参数错误（例如时间格式错误、`rejectItems` 为空）
- `401`：未登录或 Token 无效
- `404`：车辆不存在
