# 4.1.1 全线状态可视化接口

接口路径：`GET /api/v1/section/visualization/mainline`

## 请求参数
| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `timestamp` | long | 是 | 毫秒时间戳 |
