# 4.1.3 事件检测信息接口

接口路径：`GET /api/v1/event/detection/info`

## 请求参数
| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `timestamp` | long | 是 | 毫秒时间戳 |

## 返回示例
```json
{
  "code": 200,
  "success": true,
  "message": "事件检测信息数据",
  "data": {
    "timestamp": 1711958400000,
    "recordList": [
      {
        "eventId": "186300001",
        "time": "2026/03/31 10:20:11",
        "direction": "哈密",
        "eventLocation": "K3179+200",
        "eventType": "停车",
        "vehiclePlateNumber": "新A12345",
        "warningStatus": "未处理"
      }
    ]
  }
}
```

## 字段说明
- `eventId`：事件ID，优先取 `uuid`，为空时回退 `trajId`
- `time`：事件发生时间
- `direction`：根据事件里程匹配 `road_segment_static.direction`
- `eventLocation`：事件桩号位置（如 `K3179+200`）
- `eventType`：`逆行` / `停车` / `低速` / `超速` / `占道` / `其他`
- `warningStatus`：`已处理` / `未处理` / `未知`
